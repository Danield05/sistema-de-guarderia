# Sistema de Guardería

Este proyecto es un sistema de gestión diseñado para facilitar la administración de guarderías. Incluye funcionalidades para el registro de niños, padres, personal y actividades diarias.

## Tecnologías Utilizadas
- Lenguaje: PHP
- Base de datos: MySQL
- Servidor: XAMPP (Apache, MySQL, PHP)

## Instalación
1. Asegúrate de tener XAMPP instalado y ejecutándose.
2. Clona el repositorio en la carpeta `htdocs` de XAMPP (c:/xampp/htdocs/).
3. Importa la base de datos desde el archivo SQL incluido (si existe).
4. Accede al proyecto desde tu navegador en `http://localhost/sistema-de-guarderia`.
